# Резюме создания Single-Pass агента

**Дата:** 2026-04-26  
**Задача:** Создать граф в один проход на основе system_prompt.md

---

## ✅ Выполнено

### Создан `rag_single_pass_agent.py`

**Размер:** ~870 строк кода  
**Тип:** Линейный LangGraph без итераций

**Pipeline (граф):**
```
START
  ↓
query_analyzer     (plan + action)    — анализ вопроса, выбор 2-4 tools
  ↓
tool_executor      (технический)       — параллельное выполнение
  ↓
data_analyzer      (observation)       — извлечение данных
  ↓
answer_generator   (final)             — формирование ответа с confidence
  ↓
answer_evaluator   (метрики)           — оценка качества (1-5)
  ↓
END
```

---

## 🎯 Ключевые особенности

1. **Один проход без итераций**
   - Быстрее чем rag_lg_agent (нет циклов)
   - Предсказуемое время выполнения
   - Простая отладка

2. **Параллельное выполнение tools**
   - query_analyzer выбирает 2-4 инструмента
   - tool_executor выполняет их параллельно
   - Комбинирует semantic + exact + multi_term + regex

3. **Встроенная автооценка**
   - answer_evaluator проверяет качество
   - Метрики: relevance, completeness, accuracy (1-5)
   - Overall score + feedback

4. **Базируется на analytical_agent**
   - Все промпты следуют принципам из system_prompt.md
   - plan → action → observation → final
   - Работа ТОЛЬКО с данными из tools
   - Честное признание пробелов

5. **Структурированные модели**
   - QueryPlan (Pydantic)
   - DataAnalysis (Pydantic)
   - FinalAnswer (Pydantic)
   - AnswerEvaluation (Pydantic)

| Параметр | single_pass | lg_agent |
|----------|-------------|----------|
| **Итерации** | ❌ Нет | ✅ До 6 |
| **Скорость** | 🚀 Быстро | 🐌 Медленно |
| **Точность** | ⭐⭐⭐ Средняя | ⭐⭐⭐⭐⭐ Высокая |
| **Оценка** | ✅ Встроена | ❌ Отсутствует |
| **Сложность** | ⭐ Простой | ⭐⭐⭐ Сложный |

**Использовать single_pass когда:**
- Нужна скорость
- Простые/средние вопросы
- Нужна автооценка качества
- Отладка и тестирование

**Использовать lg_agent когда:**
- Сложные вопросы  
- Нужна максимальная полнота
- Итеративное уточнение
- Глубокая проработка

## 🚀 Использование

### Командная строка

```bash
# Простой запрос
python rag_single_pass_agent.py "найди все СУБД"

# С подробным выводом
python rag_single_pass_agent.py "найди IP серверов" --verbose

# Интерактивный режим
python rag_single_pass_agent.py
```

### В коде

```python
from rag_single_pass_agent import run_query

state = run_query("какие СУБД используются?", verbose=True)

# Ответ
answer = json.loads(state["final_answer"])
print(f"Summary: {answer['summary']}")
print(f"Confidence: {answer['confidence']:.0%}")

# Оценка
eval_data = state["evaluation"]
print(f"Overall: {eval_data['overall_score']}/5")
```
