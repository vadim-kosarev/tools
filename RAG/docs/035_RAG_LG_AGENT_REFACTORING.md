## 📊 Архитектура

### Новый граф (линейный)

```
START → plan_node → action_node → observation_node → final_node → END
```

### Узлы

#### 1. plan_node
- Загружает `system_prompt.md`
- Анализирует вопрос
- Возвращает JSON: `{"status": "plan", "step": 1, "thought": "...", "plan": [...]}`
- **Логируется:** system prompt + user message + assistant response

#### 2. action_node  
- Выбирает 2-4 инструмента
- Выполняет их параллельно
- Возвращает JSON: `{"status": "action", "step": 2, "thought": "...", "action": [...]}`
- **Логируется:** system prompt + assistant response + все tool calls и results

#### 3. observation_node
- Анализирует результаты
- Извлекает факты
- Возвращает JSON: `{"status": "observation", "step": 3, "thought": "...", "observation": "..."}`
- **Логируется:** system prompt + assistant response

#### 4. final_node
- Формирует итоговый ответ
- Возвращает JSON: `{"status": "final", "step": 4, "final_answer": {...}}`
- **Логируется:** system prompt + assistant response
