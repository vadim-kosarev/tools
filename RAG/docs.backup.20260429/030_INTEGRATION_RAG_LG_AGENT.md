## 🎯 Ключевые принципы (интегрированы во все промпты)

### Из analytical_agent:

1. **Итеративный подход**: plan → action → observe → conclude
2. **Работа ТОЛЬКО с tools**: запрещено придумывать данные
3. **Честность**: явное признание отсутствия данных
4. **Точность**: каждое утверждение должно иметь источник
5. **Структурированность**: строгий JSON для всех ответов (кроме finalizer)

---

## 📊 Соответствие узлов и статусов

```
analytical_agent         LangGraph узлы
────────────────────     ──────────────────────────
status: plan        →    planner
status: action      →    tool_selector + tool_executor
status: observation →    analyzer
(решение)           →    refiner
status: final       →    finalizer
```

## 🔧 Технические детали

### Промпты теперь содержат:

**Для planner:**
```
ОСНОВНОЙ ПРИНЦИП (из analytical_agent):
  - Ты НЕ отвечаешь сразу
  - Ты работаешь итеративно: plan → action → observe → conclude
  - Работа ТОЛЬКО с информацией из инструментов (tools)
  - ЗАПРЕЩЕНО придумывать, домысливать или использовать общие знания
```

**Для tool_selector:**
```
ОСНОВНОЙ ПРИНЦИП (из analytical_agent):
  - Статус: action - выбор и вызов инструмента
  - Работай ТОЛЬКО с доступными tools
  - Каждое действие должно быть обосновано
  - Избегай повторных запросов
```

**Для analyzer:**
```
ОСНОВНОЙ ПРИНЦИП (из analytical_agent):
  - Статус: observation - обработка результатов инструмента
  - Извлекай ТОЛЬКО информацию из результата инструмента
  - ЗАПРЕЩЕНО придумывать, домыс ливать или добавлять от себя
  - Честно признавай отсутствие данных
```

**Для finalizer:**
```
ОСНОВНОЙ ПРИНЦИП (из analytical_agent):
  - Статус: final - формирование итогового ответа
  - Работа ТОЛЬКО с найденными данными (found_items, evidence)
  - ЗАПРЕЩЕНО придумывать, домысливать или использовать общие знания
  - Честное признание недостаточности данных

ФОРМАТ ОТВЕТА (опционально):
  - summary: краткий ответ
  - details: подробное объяснение
  - data: структурированные данные [{entity, attribute, value}]
  - sources: список источников
  - confidence: уверенность (0-1)
```
