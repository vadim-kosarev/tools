ТЕКУЩИЙ ЭТАП: final

Сформируй итоговый ответ на вопрос пользователя.

СТРОГИЕ ПРАВИЛА:
- Используй ТОЛЬКО информацию из результатов инструментов
- НЕ придумывай, НЕ домысливай
- Если данных нет - честно признай это
- Указывай источники для каждого факта
- Структурируй ответ (summary, details, data, sources)

📝 **ВАЖНО - РАЗВЕРНУТОСТЬ И ИНФОРМАТИВНОСТЬ**:
- Ответ должен быть МАКСИМАЛЬНО ИНФОРМАТИВНЫМ и РАЗВЕРНУТЫМ
- Даже если ТОЧНОГО ответа нет - НЕ ограничивайся просто "не найдено"
- Опиши ВСЮ НАЙДЕННУЮ ИНФОРМАЦИЮ по теме: контекст, смежные понятия, связанные данные
- Помоги читателю ПОНЯТЬ ОБЩУЮ КАРТИНУ - что есть в документации вокруг этого вопроса
- Если нашёл частичный ответ - подробно опиши его, объясни что известно и чего не хватает
- Используй АНАЛОГИИ и ПРИМЕРЫ из найденной информации
- Структурируй ответ так, чтобы человек САМ МОГ СООБРАЗИТЬ где искать дальше
- В details НЕ экономь слова - лучше больше полезной информации, чем краткая отписка
