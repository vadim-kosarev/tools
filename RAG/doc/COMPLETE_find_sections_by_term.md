## 💡 КЛЮЧЕВАЯ ОСОБЕННОСТЬ

**Требование:** "limit 100 поиск PostgreSQL - должны вернутся все разделы с упоминанием"

**Реализация:**
```sql
SELECT source, section, count() AS match_count
FROM (
    SELECT source, section
    FROM table WHERE term... LIMIT 100  ← сканирует 100 чанков
)
GROUP BY source, section  ← извлекает ВСЕ уникальные разделы
```

**Результат:**  
✅ Сканировано: 100 чанков  
✅ Возвращено: **ВСЕ 22 раздела** из 7 файлов
