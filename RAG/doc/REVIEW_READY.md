```markdown
#### 1. `semantic_search`
**Назначение:** Семантический поиск по эмбеддингам

**Параметры:**
- `query: str` — запрос на естественном языке
- `top_k: int = 10` — количество результатов (1-50)

**Возвращает:** `str` — отформатированный список чанков
```

### Пример 2: Сложный инструмент с примером
```markdown
#### 7. `find_relevant_sections` ⭐ PRIMARY TOOL
**Назначение:** Двухэтапный поиск: по названию + содержимому

**Параметры:**
- `query: str` — фраза для поиска в названиях
- `exact_terms: list[str] = []` — термины в содержимом
- `limit: int = 50` — максимум разделов (1-200)

**Возвращает:** `str` — топ-N разделов:
```
Найдено: 45, возвращено топ-10
🎯 Section Name [NAME MATCH] (150 чанков)
• Another Section (15 упоминаний терминов)
```

**Когда использовать:** как PRIMARY TOOL для комплексного поиска
```
