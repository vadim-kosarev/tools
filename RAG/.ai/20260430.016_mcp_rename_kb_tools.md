## ✅ Итог

Все MCP скрипты переименованы с префиксом `kb_tools_` для консистентности с проектом.

Рекомендуемый способ использования — **stdio транспорт** (без отдельного сервера):

```yaml
# ~/.continue/config.yaml
name: KB Tools MCP
version: 0.0.1
schema: v1
mcpServers:
  - name: kb-tools
    command: C:/dev/github.com/vadim-kosarev/tools.0/RAG/kb_tools_mcp_stdio.bat
```
