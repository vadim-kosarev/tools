## 📋 Задача

Централизовать все промпты в одном файле `prompts.py` для удобного редактирования и использования IDE Find Usages.

**Требования:**
1. Все промпты в одном Python файле
2. Методы типа `get_КАКОЙ_НИБУДЬ_prompt(state)` генерируют готовый текст
3. Методы связаны с использованием через IDE Find Usages

---

## ✅ Выполнено

### 1. **Создан файл `prompts.py`**

Структура файла:
- **`get_system_prompt_base()`** - базовый системный промпт (перенесено из system_prompt.md)
- `get_plan_system_prompt(system_prompt)` - system message для plan_node
- `get_plan_user_prompt(user_query)` - user message для plan_node
- `get_plan_retry_prompt()` - retry prompt для plan_node
- `get_action_system_prompt(system_prompt, state, iteration, MAX_ITERATIONS)` - system message для action_node
- `get_action_user_prompt(user_query)` - user message для action_node
- `get_action_retry_prompt(state)` - retry prompt для action_node
- `get_observation_system_prompt(system_prompt, state, iteration, MAX_ITERATIONS)` - system message для observation_node
- `get_observation_user_prompt(user_query, state, iteration, tools_json)` - user message для observation_node
- `get_observation_retry_prompt(state)` - retry prompt для observation_node
- `get_refine_system_prompt(system_prompt, state, iteration, MAX_ITERATIONS)` - system message для refine_node
- `get_refine_user_prompt(user_query, state)` - user message для refine_node
- `get_refine_retry_prompt(state)` - retry prompt для refine_node
- `get_final_system_prompt(system_prompt, state, total_tools)` - system message для final_node
- `get_final_user_prompt(user_query, state, total_tools, all_results_json)` - user message для final_node

**Каждая функция содержит docstring с информацией "Используется в:"** - это обеспечивает работу Find Usages в IDE.

**✅ ВАЖНОЕ ОБНОВЛЕНИЕ:** Содержимое `system_prompt.md` (527 строк) полностью перенесено в `prompts.py` как функция `get_system_prompt_base()`. Файл `system_prompt.md` переименован в `system_prompt.md.backup` для сохранения истории.

## 🎯 Использование IDE Find Usages

### Как найти где используется промпт:

1. Открыть `prompts.py`
2. Навести курсор на функцию (например `get_plan_system_prompt`)
3. Нажать **Alt+F7** (или правый клик → Find Usages)
4. Увидеть список всех мест где вызывается эта функция

### Как изменить промпт:

1. Открыть `prompts.py`
2. Найти нужную функцию (например `get_plan_system_prompt`)
3. Отредактировать текст промпта внутри функции
4. Сохранить - изменения автоматически применятся везде где используется

- ✅ Все промпты в одном месте
