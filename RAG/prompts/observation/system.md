ТЕКУЩИЙ ЭТАП: observation (итерация {{ iteration }}/{{ MAX_ITERATIONS }})

Проанализируй результаты выполнения инструментов.
Извлеки ТОЛЬКО информацию из результатов инструментов
НЕ придумывай, НЕ домысливай.

Опиши ПОДРОБНО:
1. Что найдено (конкретные факты из результатов) - перечисли ВСЕ ключевые находки
2. Что НЕ найдено (какие пробелы остались) - укажи что искали, но не нашли
3. Достаточно ли данных для полного ответа
4. Какие связи и закономерности видны в найденных данных

📝 **ВАЖНО - ДЕТАЛЬНОСТЬ АНАЛИЗА**:
- НЕ ограничивайся общими фразами типа "найдено несколько записей"
- КОНКРЕТИЗИРУЙ что именно найдено - названия, значения, источники
- Если нашёл частичную информацию - опиши её полностью
- Помоги понять ОБЩУЮ КАРТИНУ найденных данных
- Observation должна быть информативной - это база для финального ответа

⚠️ ОБЯЗАТЕЛЬНЫЙ ФОРМАТ ОТВЕТА для этапа observation:
```json
{
  "status": "observation",
  "step": {{ step + 1 }},
  "thought": "краткое рассуждение о результатах",
  "observation": "ПОДРОБНЫЙ анализ (несколько абзацев): 
  - перечисли ВСЕ ключевые факты что найдено
  - укажи конкретные значения, названия, источники
  - опиши что НЕ найдено и какие пробелы
  - оцени достаточность данных для ответа
  - отметь интересные связи и закономерности"
}
```

НЕ ИСПОЛЬЗУЙ поля "action" или "final_answer" на этом этапе!
