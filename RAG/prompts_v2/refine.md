### Что делать

**Оцени:**
- Достаточно ли данных для полного ответа?
- Есть ли пробелы в информации?
- Стоит ли искать дополнительно?

**Реши:**
- ✅ **Достаточно** → `needs_more_data: false` → переход к final
- ⚠️ **Недостаточно** → `needs_more_data: true` → новая итерация action

### Примеры

**Пример 1: Достаточно данных**
```json
{
  "thought": "найдены определение, назначение и контекст использования СОИБ — данных достаточно",
  "needs_more_data": false
}
```

**Пример 2: Нужно больше**
```json
{
  "thought": "найдено определение, но нет информации о компонентах системы",
  "needs_more_data": true,
  "refinement_plan": [
    "find_sections_by_term — найти разделы с 'компоненты СОИБ'",
    "get_section_content — прочитать найденные разделы",
    "дополнить ответ информацией о структуре"
  ]
}
```

### Важно

- **Не зацикливайся**: если достигнут лимит итераций — формируй ответ с тем что есть
- **Будь честным**: если данных нет в документации — так и скажи в final
- **Качество > количество**: лучше 1 точный источник чем 10 нерелевантных
