## ЭТАП: REFINER (Принятие решения о продолжении)

Твоя задача: **проанализировать выполнение текущего шага плана** и **решить** продолжать ли выполнение плана или перейти к формированию финального ответа.

### Контекст

- **План поиска:** содержит {{ plan|length }} шагов
- **Текущий шаг:** {{ current_step_index + 1 }}/{{ plan|length }} - "{{ current_step }}"
- **Итерация:** {{ iteration }}/{{ MAX_ITERATIONS }}
- **Собрано результатов:** {{ total_results }}

### Твои действия

1. **Проанализируй выполнение текущего шага:**
   - Были ли найдены данные для текущего шага?
   - Достаточно ли информации для выполнения этого шага?
   - Остались ли невыполненные части текущего шага?

2. **Оцени общий прогресс:**
   - Сколько шагов плана уже выполнено?
   - Остались ли невыполненные шаги?
   - Достаточно ли собрано данных для ответа на вопрос пользователя?

3. **Прими решение:**
   - **continue** - если есть невыполненные шаги плана И iteration < {{ MAX_ITERATIONS }}
   - **final** - если все критичные шаги выполнены ИЛИ iteration >= {{ MAX_ITERATIONS }} ИЛИ собрано достаточно данных для ответа

### Критерии принятия решения

**Продолжить (continue):**
- ✅ Текущий шаг выполнен, но есть еще шаги в плане
- ✅ Текущий шаг выполнен частично, нужно дополнить
- ✅ Iteration < {{ MAX_ITERATIONS }}

**Завершить (final):**
- ✅ Все шаги плана выполнены
- ✅ Собрано достаточно данных для полного ответа
- ✅ Iteration >= {{ MAX_ITERATIONS }} (лимит достигнут)
- ✅ Дальнейший поиск не даст новой информации
