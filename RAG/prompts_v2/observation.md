
## ЭТАП: OBSERVATION (Анализ результатов)

### 🎯 ВОПРОС ПОЛЬЗОВАТЕЛЯ (РЕАЛЬНЫЙ):

```
{{ user_query }}
```

⚠️ **Анализируй результаты** относительно ЭТОГО вопроса.

### Результаты инструментов

```json
{{ tools_results }}
```

### Что делать

1. **Изучи результаты:**
   - Что нашли инструменты?
   - Релевантна ли информация?
   - Достаточно ли данных?

2. **Проанализируй:**
   - Есть ли ответ на вопрос?
   - Нужна ли дополнительная информация?
   - Какие факты извлечь?

3. **Сформируй observation:**
   - Краткое резюме найденного
   - Оценка полноты
   - Идеи для уточнения (если нужно)

### Формат ответа

```json
{{ schema_AgentObservation }}
```

**НЕ используй** поля `actions`, `final_answer` на этом этапе.

###Пример

```json
{
  "thought": "найдены определения СОИБ в нескольких разделах, данных достаточно",
  "observation": "Semantic search нашел 5 релевантных чанков с определением СОИБ КЦОИ. Exact search добавил 12 упоминаний термина в контексте. Основные источники: 'Общее описание системы.md' и 'Техническое задание.md'. Определение четкое: 'Система обеспечения информационной безопасности Комплекса центров обработки информации'. Данных достаточно для полного ответа."
}
```
