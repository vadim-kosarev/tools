# 20260428.009 - Расширенная стратегия определения терминов для поиска

**Дата:** 2026-04-28  
**Автор:** AI Agent (Claude)  
**Проблема:** Агент использовал только 3-5 основных терминов без синонимов и связанных слов

## Описание проблемы

При анализе вопроса "Какое ПО установлено на АРМ оператора" агент определял только основные термины:
- ❌ "ПО", "АРМ оператора", "установлено" (всего 3-4 термина)
- ❌ Нет синонимов: "программное обеспечение", "программные средства"
- ❌ Нет связанных слов: "программа", "приложения", "компонент"
- ❌ Нет синонимов связанных слов: "утилита", "модуль", "сервис"

**Результат:** Поиск находил только документы с точными терминами из вопроса, пропуская релевантные документы с синонимами.

## Решение

Обновлены промпты для агента с инструкцией определять **РАСШИРЕННЫЙ список терминов (10-15)**:

### Новая стратегия определения терминов (5 шагов):

1. **Шаг 1: Извлечь основные термины** из вопроса
   - "ПО", "АРМ оператора", "установлено"

2. **Шаг 2: Добавить синонимы** для КАЖДОГО термина
   - "ПО" → "программное обеспечение", "программные средства", "софт", "приложения"
   - "АРМ оператора" → "АРМ", "рабочее место", "рабочая станция"
   - "установлено" → "развернуто", "настроено", "запущено"

3. **Шаг 3: Добавить связанные слова**
   - "программа", "компонент", "система", "сервис"

4. **Шаг 4: Добавить синонимы связанных слов**
   - "программа" → "утилита", "модуль"
   - "компонент" → "элемент", "составная часть"

5. **Шаг 5: Добавить СЛОВОСОЧЕТАНИЯ** из запроса и их комбинации
   - Словосочетания из запроса: "ПО установлено", "установлено на АРМ", "ПО на АРМ"
   - Комбинации с синонимами: "программное обеспечение установлено", "софт на рабочем месте"
   - Распространенные фразы: "установлено следующее ПО", "список установленного ПО"

- ✅ Определить РАСШИРЕННЫЙ список (15-20 терминов)
- ✅ Синонимы каждого термина
- ✅ Связанные слова
- ✅ Синонимы связанных слов
- ✅ Словосочетания из запроса и их комбинации

**Обновлена инструкция по планированию:**

```markdown
**При планировании учитывай тип вопроса:**

**Важно для multi_term_exact_search:**

- Все синонимы каждого термина
- Связанные слова и понятия
- Синонимы связанных слов
```

## Преимущества нового подхода

### 1. Лучшее покрытие документации

**Пример:** Документ содержит "программные средства установлены на рабочей станции"
- ❌ Старый подход: НЕ НАЙДЕНО (нет точных терминов "ПО", "АРМ оператора", "установлено")
- ✅ Новый подход: НАЙДЕНО (есть "программные средства", "рабочей станции", "установлены")

### 2. Устойчивость к вариациям формулировок

Документация может использовать разные термины:
- "ПО" / "программное обеспечение" / "программные средства" / "софт"
- "АРМ" / "рабочее место" / "рабочая станция"
- "установлено" / "развернуто" / "настроено"

Новый подход покрывает ВСЕ варианты.

### 3. Связанные понятия

Если вопрос про "ПО", то релевантны документы с:
- "программа", "приложение", "компонент"
- "утилита", "модуль", "сервис"

Новый подход автоматически ищет и эти термины.

### 4. Словосочетания из запроса

Документы часто содержат целые фразы, а не отдельные слова:
- "ПО установлено на АРМ"
- "программное обеспечение установлено на рабочем месте"
- "список установленного ПО"

Поиск по словосочетаниям **значительно повышает точность** нахождения релевантных документов.
