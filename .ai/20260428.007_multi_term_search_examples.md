# 20260428.007 - Добавление примеров использования multi_term_exact_search в промпты

**Дата:** 2026-04-28  
**Автор:** AI Agent (Claude)  
**Проблема:** Агент использовал `exact_search` только с одним термином вместо `multi_term_exact_search` с массивом терминов

## Описание проблемы

При анализе логов обнаружено, что агент для вопроса "Какое ПО установлено на АРМ оператора":
1. Определил стратегию поиска, но упомянул только `exact_search`, а не `multi_term_exact_search`
2. Вызвал `exact_search` только с ОДНИМ термином "установлены следующие"
3. Не выделил и не использовал ключевые термины из вопроса: "АРМ оператора", "ПО", "программное обеспечение"

### Файл лога с проблемой
```
RAG/logs/008_tool_exact_search_request.log:
substring='установлены следующие'
limit=30
```

### Причина
В промптах было **упоминание** инструмента `multi_term_exact_search`, но **НЕТ конкретных примеров** его использования с массивом терминов.

## Внесенные изменения

### 1. Базовый системный промпт (system_prompt_base.md)

**Добавлено:**
- ✅ Расширенный раздел "Для точного поиска по нескольким терминам" с пометкой "ПРИОРИТЕТНАЯ СТРАТЕГИЯ"
- ✅ Подробные инструкции:
  * Когда использовать (вопросы типа "Какое X на Y")
  * Правильный подход: определить 3-5 ключевых терминов
  * Пример определения терминов для вопроса "Какое ПО установлено на АРМ оператора"
- ✅ Блок "НЕПРАВИЛЬНО / ПРАВИЛЬНО" с конкретными примерами ошибок
- ✅ Новый пример JSON с `multi_term_exact_search` и массивом терминов:
  ```json
  {
    "tool": "multi_term_exact_search",
    "input": {
      "terms": ["АРМ оператора", "программное обеспечение", "установлено", "ПО"],
      "limit": 50
    }
  }
  ```

### 2. Промпт для action node (action/system.md)

**Добавлено:**
- ✅ Пример параметров `multi_term_exact_search` в список примеров инструментов:
  ```
  - multi_term_exact_search: {"terms": ["термин1", "термин2", "термин3"], "limit": 50}
  ```

### 3. Промпт для plan node (plan/system.md)

**Добавлено:**
- ✅ Инструкции по выбору инструментов в зависимости от типа вопроса:
  ```
  При планировании учитывай тип вопроса:
  - Для вопросов "Какое X установлено на Y" → определи 3-5 ключевых терминов → используй multi_term_exact_search
  - Для вопросов "Что такое X" → используй semantic_search + exact_search
  - Для вопросов "Дай список/перечень" → используй find_sections_by_term + get_section_content
  ```

## Ожидаемый результат

После изменений агент будет:
