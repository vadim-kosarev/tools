# Добавление поддержки цифр в аббревиатурах (find_abbreviation_expansion)

**Дата:** 2026-04-29  
**Номер:** 20260429.002  
**Модуль:** RAG/kb_tools.py  
**Тип изменения:** Улучшение функционала

---

## Краткое описание

Добавлена поддержка **цифр** в аббревиатурах для инструмента `find_abbreviation_expansion`. 

Теперь инструмент может искать расшифровки аббревиатур с цифрами, например:
- `AK47` → "Автомат Калашникова 47"
- `T34` → "Танк 34"
- `MP5` → "Maschinenpistole 5"

---

## Что изменено

### 1. Функция `_build_abbreviation_pattern`

**Было:**
```python
# Определение языка только по первой букве
# Ошибка при любых нелатинских/некириллических символах
# Все символы должны быть буквами
for char in abbreviation.upper():
    if not char.isalpha():
        raise ValueError(f"Abbreviation must contain only letters: {abbreviation}")
    parts.append(f"{char}{letter_class}*")
```

**Стало:**
```python
# Динамическое определение типа каждого символа
while i < len(abbreviation):
    char = abbreviation[i]
    
    if char.isdigit():
        # Собираем последовательность цифр
        digit_sequence = char
        i += 1
        while i < len(abbreviation) and abbreviation[i].isdigit():
            digit_sequence += abbreviation[i]
            i += 1
        # Цифры - точное совпадение
        parts.append(digit_sequence)
    
    elif char.isalpha():
        # Определяем алфавит для ЭТОЙ буквы
        upper_char = char.upper()
        if 'А' <= upper_char <= 'Я' or upper_char == 'Ё':
            letter_class = r'[а-яё]'
        elif 'A' <= upper_char <= 'Z':
            letter_class = r'[a-z]'
        parts.append(f"{upper_char}{letter_class}*")
        i += 1
```

### 2. Улучшения алгоритма

**Ключевые изменения:**
1. **Посимвольная обработка** - каждый символ обрабатывается индивидуально
2. **Группировка цифр** - последовательные цифры объединяются (47, не 4 и 7)
3. **Динамическое определение алфавита** - для каждой буквы отдельно
4. **Точное совпадение цифр** - цифры не экспандятся как буквы

## Технические детали

### Алгоритм обработки цифр

```python
if char.isdigit():
    # Собираем ВСЕ последовательные цифры
    digit_sequence = char
    i += 1
    while i < len(abbreviation) and abbreviation[i].isdigit():
        digit_sequence += abbreviation[i]
        i += 1
    # Добавляем как есть (точное совпадение)
    parts.append(digit_sequence)
```

**Результат:**
- `47` → `47` (не `4[a-z]*\s+7[a-z]*`)
- `249` → `249` (не `2[a-z]*\s+4[a-z]*\s+9[a-z]*`)

### Алгоритм обработки букв

```python
elif char.isalpha():
    upper_char = char.upper()
    # Определяем алфавит ДЛЯ ЭТОЙ буквы
    if 'А' <= upper_char <= 'Я' or upper_char == 'Ё':
        letter_class = r'[а-яё]'
    elif 'A' <= upper_char <= 'Z':
        letter_class = r'[a-z]'
    parts.append(f"{upper_char}{letter_class}*")
    i += 1
```

**Преимущество:** В будущем можно поддерживать `КЦOI` (кириллица + латиница).

## Использование

### Пример 1: Военная техника

```python
from kb_tools import create_kb_tools

tools = create_kb_tools(vectorstore, knowledge_dir)
find_abbr = next(t for t in tools if t.name == "find_abbreviation_expansion")

result = find_abbr.invoke({"abbreviation": "AK47"})
# Результат: {"expansions": ["Автомат Калашникова 47"], "total_found": 1}
```
