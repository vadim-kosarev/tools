## Краткое описание

Изменен вывод команды `python kb_tools.py list` с текстового списка на **табличный формат** для лучшей читаемости.

## Преимущества

1. **Компактность**
   - Все данные на одном экране
   - Легко сканировать глазами
   - Не нужно скроллить

2. **Читаемость**
   - Четкие границы колонок
   - Выравнивание данных
   - Профессиональный вид

3. **Информативность**
   - Видно количество инструментов
   - Подсказка как получить детали
   - Параметры сразу в одной строке

## Реализация

### Функция `_cli_list_tools`

```python
def _cli_list_tools(tools: list[BaseTool]) -> None:
    """Выводит краткий список инструментов в табличном формате"""
    
    # Собираем данные
    rows = []
    for tool in tools:
        param_names = []
        # ... сбор параметров ...
        params_str = ', '.join(param_names) if param_names else '-'
        rows.append((tool.name, params_str))
    
    # Вычисляем ширину колонок
    max_name_len = max(len(row[0]) for row in rows)
    max_params_len = max(len(row[1]) for row in rows)
    
    name_width = max(max_name_len, len("Инструмент"))
    params_width = max(max_params_len, len("Параметры"))
    
    # Рисуем таблицу
    separator = f"+{'-' * (name_width + 2)}+{'-' * (params_width + 2)}+"
    
    print(separator)
    print(f"| {'Инструмент':<{name_width}} | {'Параметры':<{params_width}} |")
    print(separator)
    
    for name, params in rows:
        print(f"| {name:<{name_width}} | {params:<{params_width}} |")
    
    print(separator)
    print(f"\nВсего инструментов: {len(rows)}")
    print("\nДля детальной справки используйте: python kb_tools.py help <инструмент>")
```

**Ключевые особенности:**
- Автоматический расчет ширины колонок
- Учет минимальной ширины заголовков
- ASCII-art разделители
- Итоговая информация

## Формат таблицы

### Структура

```
+----------+------------+
| Колонка1 | Колонка2   |
+----------+------------+
| данные   | данные     |
| данные   | данные     |
+----------+------------+
```

### Колонки

1. **Инструмент** - название инструмента
2. **Параметры** - список параметров (обязательные и [опциональные])

### Обозначения

- `param` - обязательный параметр
- `[param]` - опциональный параметр
- `-` - нет параметров
